voice-to-us widget

Three drop-in modes. Pick one, paste the snippet into any site, done.

1. Inline — auto-mount

<script
  src="https://cdn.jsdelivr.net/gh/gmic-ai-inc/voice-to-us@v0.1.0/widget/voice-to-us.js"
  data-backend="https://gmic.ai/voice2us"
  data-mount="#voice-btn"></script>
<div id="voice-btn"></div>

2. Programmatic

<script src="https://cdn.jsdelivr.net/gh/gmic-ai-inc/voice-to-us@v0.1.0/widget/voice-to-us.js"></script>
<div id="voice-btn-2"></div>
<script>
  VoiceToUs.mount('#voice-btn-2', { backend: 'https://gmic.ai/voice2us' });
</script>

3. Floating bubble

<script
  src="https://cdn.jsdelivr.net/gh/gmic-ai-inc/voice-to-us@v0.1.1/widget/voice-to-us.js"
  data-backend="https://gmic.ai/voice2us"
  data-floating="true"
  data-placement="bottom-center"></script>

data-placement accepts bottom-right (default), bottom-center, or bottom-left. Try it live:

4. Custom styling

Override colors, size, and labels via data-* attributes or the theme / labels options.

<script
  src="https://cdn.jsdelivr.net/gh/gmic-ai-inc/voice-to-us@v0.1.0/widget/voice-to-us.js"
  data-backend="https://gmic.ai/voice2us"
  data-mount="#voice-btn-pink"
  data-color="#e91e63"
  data-ring-pulse-color="#f48fb1"
  data-size="88"
  data-label-idle="Send us a voice note"></script>

Three instances above use pink / blue / minimal themes. See the page source.
