Three-line embed

One script tag, one placeholder div. No framework required.

Shadow DOM isolated

Host page styles can't leak in. Widget styles can't leak out.

Fully themeable

Colors, size, position, labels — all via data-* attributes.

Context attached

01 Inline auto-mount

The simplest path. Add the script tag, add a placeholder div, done.

<script
  src="https://cdn.jsdelivr.net/gh/gmic-ai-inc/voice-to-us@v0.1.9/widget/voice-to-us.js"
  data-backend="https://gmic.ai/voice2us"
  data-mount="#voice-btn"
  async></script>
<div id="voice-btn"></div>
LIVE

02 Programmatic mount

For SPAs or when the placeholder renders after the page loads.

<script src="https://cdn.jsdelivr.net/gh/gmic-ai-inc/voice-to-us@v0.1.9/widget/voice-to-us.js"></script>
<div id="voice-btn"></div>
<script>
  VoiceToUs.mount('#voice-btn', {
    backend: 'https://gmic.ai/voice2us',
  });
</script>
LIVE

03 Floating bubble

Pinned to a corner or the bottom center, like a chat widget. No placeholder needed.

<script
  src="https://cdn.jsdelivr.net/gh/gmic-ai-inc/voice-to-us@v0.1.9/widget/voice-to-us.js"
  data-backend="https://gmic.ai/voice2us"
  data-floating="true"
  data-placement="bottom-center"
  async></script>

04 Custom styling

Override colors, size, and labels via data-* attributes or the theme / labels option.

<script
  src="https://cdn.jsdelivr.net/gh/gmic-ai-inc/voice-to-us@v0.1.9/widget/voice-to-us.js"
  data-backend="https://gmic.ai/voice2us"
  data-mount="#voice-btn-pink"
  data-color="#e91e63"
  data-ring-pulse-color="#f48fb1"
  data-size="88"
  data-label-idle="Send us a voice note"></script>

05 Custom tip text (all states)

Every status message is overridable — for localization, brand voice, or whatever tone fits your page.

<script

  data-backend="https://gmic.ai/voice2us"
  data-mount="#voice-btn-labels"
  data-label-idle="Leave us a voice note"
  data-label-recording="Listening… tap to finish"
  data-label-uploading="Delivering…"
